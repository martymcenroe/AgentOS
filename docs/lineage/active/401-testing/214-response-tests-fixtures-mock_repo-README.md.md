# Mock Project

A sample project for testing codebase context analysis in AssemblyZero.

## Features

- User authentication via `src/auth.py`
- Main application entry point in `src/main.py`
- Poetry-based dependency management

## Getting Started

Install dependencies with `poetry install` and run with `poetry run python src/main.py`.

## Architecture

The project follows a simple modular structure with source code in `src/` and tests in `tests/`.
