# Mock Repository

A sample Python project for testing codebase context analysis.

## Overview

This is a mock repository used by the AssemblyZero test suite to verify
that the codebase analysis node correctly reads project files, detects
patterns, and extracts conventions.

## Features

- Authentication module with user login/logout
- Main application entry point
- Poetry-based dependency management

## Getting Started

```bash
poetry install
poetry run python src/main.py
```
